华为移动路由PRO（B535-836）实际试用的一些测试

· 2022-10-27 · # 路由器

几个关键参数

RSRQ（Reference Signal Receiving Quality，参考信号接收质量）

https://baike.baidu.com/item/RSRQ/8266810
范围：-3~-19.5
绝对值越小越好

RSRP (Reference Signal Receiving Power，参考信号接收功率）

https://baike.baidu.com/item/RSRP/9967275
范围：
Rx > -65 最好
-75< Rx <= -65 好
-85 < Rx <= -75 一般
-95 < Rx <= -85 差

RSSI（Received Signal Strength Indicator，接收信号强度指示）

https://baike.baidu.com/item/rssi/10924837

一般在-50dBm~-90dBm
-60dBm以上属于信号好

SINR（Signal to Interference plus Noise Ratio，信号与干扰加噪声比（信噪比））

https://baike.baidu.com/item/SINR/5821731
TD-LTE局点，99%区域，SINR>-3dB
外场，99.25%区域，SINR>-3dB
