MacOS原生「便签」应用体验

· 2021-08-23 · # 日常

便签几乎是桌面端必备应用，一个好用的“便签”可以解决很多复杂化的问题，最近一段时间试用了Mac中的便签应用，五味杂陈……

这几天体验了下「便签」这款MacOS原生应用，整理了一些使用体验：

  1. 便签功能对于排版的兼容性比较差，当从其他位置（如Typora、网页）复制内容时，格式中会出现白色底且无论如何也无法消除，这种情况下粘贴时需要使用“粘贴并匹配样式”；
  2. 从便签中复制内容到其他位置，将失去大部分格式，不便于格式继承，但可以将便签内容复制到MacOS备忘录，大部分格式都可以得到保留；
  3. 便签支持将所有便签全部倒入导入备忘录，备忘录中会按照便签颜色自动分类保存，方便整理；
  4. 便签不支持Markdown等类似语法，需要掌握一些快捷键才能实现简单的排版能力；
  5. 便签支持半透明显示，不干扰视线，这个体验很好；
  6. 便签支持缩小为一行，适合多便签叠放管理。
